# 🛒 NexRec — AI Product Recommendation System 🚀

> Smart AI recommendations for better shopping experiences 🤖📊

## 📌 Overview
NexRec is an AI-powered E-Commerce Analytics platform that recommends products based on customer behavior, purchase history, ratings, and trends. The project combines Machine Learning, Data Analytics, and Interactive Dashboards to create personalized shopping experiences.

---

## ✨ Features

- 🤖 AI Product Recommendation Engine
- 📊 Customer Behavior Analytics
- 📈 Sales & Revenue Insights
- 🔥 Trending Products Analysis
- ⭐ Product Rating Analytics
- 🛍️ Personalized Product Suggestions
- 🌐 Interactive Streamlit Web App
- 🗄️ SQL Database Integration

---
